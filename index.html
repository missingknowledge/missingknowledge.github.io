    Variables

    In Rust, variables are immutable by default. You declare variables using the `let` keyword:

    Example:

    let x = 42;

    This declares a variable `x` with the value `42`. By default, variables are immutable, meaning their values cannot be changed once they're bound.

    If you need mutable variables, you can use the `mut` keyword:

    let mut y = 13;

    This declares a mutable variable `y` with the initial value `13`, which can be changed later in the program.
